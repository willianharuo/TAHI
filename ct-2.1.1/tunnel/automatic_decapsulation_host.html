NAME

        automatic_decapsulation_host.seq - Verify that node 
        decapsulate IPv6 datagram in IPv4

TARGET

        Host

SYNOPSIS

        automatic_decapsulation_host.seq [-tooloption ...] -p 
        automatic_decapsulation_host.def

INITIALIZATION

	Before this test starts, run initialize.seq.

TEST PROCEDURE

        This test verifies that NUT decapsulates a IPv6 packet in 
        automatic tunnel.
        


        TN                      NUT


        ---------------------------


        1.
         === IPv4 packet encapsulating IPv6 packet ===>
                Outer Header (IPv4)     
                        src address : Off-link IPv4 address
                        dst address : NUT's IPv4 address        
                Inner Header (IPv6)     
                        src address : Off-link IPv4 compatible IPv6 address
                        dst address : NUT's IPv4 compatible IPv6 address        
        2.
        In NUT, received IPv4 packet is decapsulated, and NUT creates the 
        IPv6 Echo reply packet in response to encapsulated IPv6 Echo Reply.


        3.
        << JUDGMENT >>


         <=== IPv4 packet encapsulating IPv6 packet ===
                Outer Header (IPv4)     
                        src address : NUT's IPv4 address        
                        dst address : Off-link IPv4 address
                Inner Header (IPv6)     
                        src address : NUT's IPv6 address
                        dst address : Offlink IPv6 global address

JUDGMENT

        << PASS >>
                NUT received IPv4 packet encapsulating IPv6 packet 
                


        << FAIL >>      
                NUT did not receive IPv4 packet encapsulating IPv6 packet
