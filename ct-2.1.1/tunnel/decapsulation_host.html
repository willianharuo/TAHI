NAME

        decapsulation_host.seq - Verify that node decapsulate IPv6 
        datagram in IPv4 packet

TARGET

        Host

SYNOPSIS

        decapsulation_host.seq [-tooloption ...] -p 
        decapsulation_host.def

INITIALIZATION

	Before this test starts, run initialize.seq.

TEST PROCEDURE

        This test verifies that NUT decapsulates the IPv6 packet in a IPv4 
        packet
        


        TN                      NUT


        ---------------------------


        1.
         === IPv4 packet encapsulating IPv6 packet ===>
                Outer Header (IPv4)     
                        src address : Off-link IPv4 address
                        dst address : NUT's IPv4 address        
                Inner Header (IPv6)     
                        src address : Off-link IPv6 global address
                        dst address : NUT's IPv6 global address        
        2.
        In NUT, received IPv4 packet is decapsulated, and NUT creates the 
        IPv6 Echo reply packet in response to encapsulated IPv6 Echo Request.
        3.
        << JUDGMENT >>


         <=== ICMPv6 Echo Reply ===
                IPv6 Header
                        src address : NUT's IPv6 address
                        dst address : Offlink IPv6 global address

JUDGMENT

        << PASS >>
                NUT received ICMPv6 Echo Reply
                


        << FAIL >>      
                NUT did not receive ICMPv6 Echo Reply
