NAME

        encapsulation_router.seq - Verify that encapsulation IPv6 
        datagram in IPv4 packet

TARGET

        Router

SYNOPSIS

        encapsulation_router.seq [-tooloption ...] -p 
        encapsulation_router.def

INITIALIZATION

	Before this test starts, run initialize.seq.

TEST PROCEDURE

        This test verifies that NUT encapsulates the IPv6 packet in a
        IPv4 packet
        
        Network Topology


        Link0
        --------------------------
            |               |
           TN              NUT
            |               |
        --------------------------
        Link1


        TN -- (Link0) -- NUT        TN -- (Link1) -- NUT


        ---------------------      ------------------------


        1.
        ===================>
        ICMPv6 Echo Request
                IPv6 Header
                        src address : TN LINK0  IPv6 global address        
                        dst address : Off-link IPv6 global address


    2.
        <<JUDGMENT>>
                                   <===================
        IPv4 packet encapsulating IPv6 packet
                Outer Header (IPv4)
                        src address : NUT Link1 IPv4 address
                        dst address : Off-link IPv4 address
                Inner Header (IPv6)
                        src address : TN LINK0  IPv6 global address        
                        dst address : Off-link IPv6 global address

JUDGMENT

        << PASS >>
                NUT received IPv4 packet encapsulating IPv6 datagram
                


        << FAIL >>      
                NUT did not receive IPv4 packet encapsulating IPv6 datagram
